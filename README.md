# React Crypto Dashboard

A responsive crypto dashboard built with React, Vite, and Chakra UI.

## Features

* Responsive dashboard layout
* Portfolio overview card
* Transaction management page
* Desktop transaction table
* Mobile transaction card view
* Category-based filtering
* Search by transaction ID and destination
* Status badges for transactions
* Modern UI with Chakra UI

## Tech Stack

* React
* Vite
* Chakra UI
* React Icons
* JavaScript

## Learning Outcomes

This project helped me practice:

* Component-based architecture
* State management with React Hooks
* Responsive design
* Conditional rendering
* Array filtering and searching
* Git and GitHub workflow

## Getting Started

```bash
npm install
npm run dev
```
